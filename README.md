## Setup

Run following commands

```
% git clone --recursive git@github.com:ozgurgunes/go-design-system.git
% cd go-design-system
% ./_hooks/setup.sh
```

Your repo is ready for Git actions. Checkout anything to generate Sketch files.

```
% git checkout master
```
